Game IOT : Haunted Assembly

Game edukasi yang bertema horor ini mempunyai tujuan untuk mengumpulkan semua komponen IOT dan menjadikannya satu untuk keluar dari tempat yang menyeramkan. Anda harus mengumpulkan semua komponen IOT yang ada dan menggunakannya untuk mengaktifkan perangkat yang dapat membantu Anda keluar dari tempat itu.
Lihat lebih lanjut

1. Nama game: IOT : Haunted Assembly

2. Deskripsi produk: Game ini adalah game yang kami buat dengan sepenuh hati untuk memperkenalkan iot kepada masyarakat luas. Kami bekerja siang malam hanya untuk menyelesaikan game ini tepat waktu, kami begitu mendedikasikan waktu kami untuk game ini.

3. Fitur game :

  • - Player bisa mengambil bahan bahan iot untuk di susun
  • - Player dikejar oleh sebuah hantu ai yang terus mengejar

4. Cara bermain : Rakitlah sebuah iot penyiram tanaman otomatis sambil menghindari hantu yang mengejar

Tutorial memainkan game :

  • 1. WASD untuk bergerak
  • 2. E untuk mengambil dan menjatuhkan barang
  • 3. Shift untuk sprint
